🎓 LLM이 학습한 것
GitHub의 수십억 줄의 오픈소스 코드를 학습. "다음 토큰 맞추기"로 코드를 학습하면 → 문법, 패턴, 함수 구조, 에러 해결 방법까지 확률로 저장됨. 모든 프로그래밍 언어를 동시에 학습하므로 언어 간 번역도 가능.
🔑 에이전트인 이유
단순 코드 생성이 아닌 → 코드 실행 → 에러 확인 → 수정 → 재실행을 자율 반복. 파일 시스템 탐색, 터미널 명령 실행, 테스트 수행까지 도구를 사용하는 에이전트로 동작.

🎓 LLM이 학습한 것
멀티모달 학습: 이미지와 텍스트를 함께 학습. 화면의 스크린샷을 보고 "이건 버튼이다", "여기가 입력창이다"를 이해. UI 레이아웃, 아이콘 의미, 웹사이트 구조를 수억 장의 스크린샷에서 학습. "다음 행동 맞추기" = "이 화면 상태에서 사람이 보통 어디를 클릭하는가"를 예측.
🔑 에이전트인 이유
스크린샷 → 판단 → 행동 → 스크린샷 루프를 자율 반복. API가 없는 앱도 사람처럼 GUI로 조작 가능. ERP 데이터 입력, 엑셀 정리, 사내 시스템 조작 등 "사람만 할 수 있었던" PC 작업을 AI가 대신.
💡 "다음 단어 맞추기"와의 연결
텍스트에서 "다음 단어"를 맞추듯, 화면에서 "다음 행동"을 맞추는 것. 이전의 행동들과 현재 화면을 보고 "다음에 할 가장 적절한 행동은?"을 예측 → 이것도 결국 조건부 확률 분포에서의 샘플링. CH2에서 배운 원리가 텍스트를 넘어 시각과 행동으로 확장된 것.
CEO 시사점: API가 없는 레거시 사내 시스템도 AI가 조작 가능. 반복적인 PC 작업(데이터 입력, 양식 작성, 시스템 간 이관)을 자동화하는 가장 범용적인 방법.

🎓 LLM이 학습한 것
인터넷 전체의 텍스트를 학습하여 "어떤 정보가 어디에 있는지", "신뢰할 수 있는 소스는 무엇인지"에 대한 감각을 보유. "다음 단어 맞추기"로 학술 논문, 보고서, 기사를 학습 → 좋은 리서치 보고서의 구조와 논리 전개 패턴을 확률로 체득.
🔑 에이전트인 이유
한 번 질문하면 수십 분 동안 자율적으로 검색 → 읽기 → 분석 → 재검색 반복. 사람이 개입 없이 30~100개 이상의 소스를 탐색하고 종합. CH6 에이전트의 "자율 판단" 능력이 리서치에 적용된 것.
CEO 시사점: 경쟁사 분석, 시장 조사, 기술 동향, 규제 변화 추적. 주니어 애널리스트가 1주일 걸릴 리서치를 AI가 30분에 초안 완성. 임원 보고 자료의 기초 조사에 즉시 활용 가능.

🎓 LLM이 학습한 것
수백만 개의 데이터 분석 코드(Python/R)와 그 결과 해석을 학습. "이 데이터에서 이런 질문을 하면 → 이런 분석을 하고 → 이런 결론을 내린다"는 패턴 전체를 학습. 자연어 질문 → 코드 → 시각화 → 해석의 전체 파이프라인을 확률로 체득.
🔑 에이전트인 이유
코드 생성 → 직접 실행 → 결과 확인 → 추가 분석 자율 반복. 분석 결과를 보고 "이 패턴이 이상하니 다른 각도로 분석해보겠다"고 스스로 판단. 코드 에러가 나면 자동 수정 후 재실행.
CEO 시사점: 데이터 분석 전문가가 없어도, 엑셀 파일을 올리고 "분석해줘"라고 말하면 차트와 인사이트 완성. SQL 쿼리를 몰라도 자연어로 데이터베이스 조회 가능. 의사결정 속도가 근본적으로 달라짐.

🎓 LLM이 학습한 것
텍스트: 수조 문장에서 "다음 단어 맞추기". 이미지: 수억 장의 이미지+설명 쌍에서 "이 텍스트 → 이런 이미지" 패턴 학습 (Diffusion). 핵심: 모달리티가 다를 뿐, 결국 "다음 요소를 예측"하는 동일한 원리의 확장.
🔑 다른 모달리티, 같은 원리
텍스트: 다음 토큰 예측 → 문장.
이미지: 다음 픽셀/패치 예측 → 그림.
음악: 다음 음표/파형 예측 → 멜로디.
모두 확률분포를 학습하고 샘플링하는 동일 구조. CH1~CH2에서 배운 원리가 모든 곳에 적용됨.
CEO 시사점: 마케팅 소재(이미지·카피), 프레젠테이션, 영상 콘텐츠의 초안 생성 속도가 수십 배 빨라짐. 크리에이티브 팀의 역할이 "생성"에서 "기획+검수"로 전환.
